Delete a chore

Deletes a chore by ID. For recurring chores, use the apply_to param to control whether only this occurrence, this and following, or all occurrences are deleted. An alternative form using a date string as choreId is also observed (delete occurrence on a specific date).

Authorizations:
skylight_auth
path Parameters
frameId
required
string
choreId
required
string

Chore ID or YYYY-MM-DD date (for date-based deletion of recurring chores).

query Parameters
apply_to
string

Scope: 'this', 'this_and_following', 'all'

Responses

https://app.ourskylight.com/api/frames/{frameId}/chores/{choreId}
